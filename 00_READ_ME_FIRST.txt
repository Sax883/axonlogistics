═════════════════════════════════════════════════════════════════════════════
                      ⚙️ HOW TO USE ADMIN PANEL
═════════════════════════════════════════════════════════════════════════════

CREATE A NEW SHIPMENT:
1. Open admin/dashboard.html
2. Click "New Shipment" tab
3. Fill in sender/receiver information
4. Select origin and destination cities
5. Enter weight and delivery date
6. Click "Create Shipment"
✓ System auto-generates tracking number

UPDATE SHIPMENT STATUS:
1. Go to "Shipments" tab
2. Click Edit button (pencil icon)
3. Select new status from dropdown
4. Add update message
5. Choose current location
6. Click "Update"
✓ Status updates in real-time

MANAGE CLIENTS:
1. Go to "Clients" tab
2. Click "Add Client" button
3. Fill in client information
4. Click "Save Client"
✓ Edit or delete as needed

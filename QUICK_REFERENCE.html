Common Admin Tasks
Create Shipment
  1. Go to "New Shipment" tab
  2. Fill sender/receiver info
  3. Select origin/destination
  4. Set weight & delivery date
  5. Click "Create Shipment"
Update Status
  1. Go to "Shipments" tab
  2. Click Edit button
  3. Select new status
  4. Add update message
  5. Select location & click Update
Manage Clients
  1. Go to "Clients" tab
  2. Click "Add Client"
  3. Fill client information
  4. Click "Save Client"
  5. Edit/Delete as needed
How to Customize
Change Company Information:

Edit the AXON_HQ object in js/data.js

Change Colors:

Edit CSS variables in css/style.css (lines 1-8)

Change Logo:

Replace assets/logo.svg with your logo

Add More Cities:

Add cities to MAJOR_CITIES array in js/data.js

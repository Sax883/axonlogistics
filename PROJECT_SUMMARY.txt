## 🔧 How to Use Admin Dashboard

### Create Shipment:
1. Click "New Shipment" tab
2. Fill sender/receiver info
3. Select origin & destination cities
4. Enter weight & delivery date
5. Set initial status
6. Click "Create Shipment"

### Update Status:
1. Click "Shipments" tab
2. Click Edit button on shipment
3. Select new status
4. Add update message
5. Choose current location
6. Click "Update"

### Manage Clients:
1. Click "Clients" tab
2. Click "Add Client" button
3. Fill client information
4. Click "Save Client"
5. Edit or delete as needed

## 🎨 Customization Quick Start

### Change Company Name/Contact:
Open `js/data.js` and edit the AXON_HQ object (around line 200)

### Change Colors:
Open `css/style.css` and edit :root variables (lines 1-8)

### Change Logo:
Replace `assets/logo.svg` with your logo file

### Add More Cities:
Edit MAJOR_CITIES array in `js/data.js`
